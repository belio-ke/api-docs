<Note>
  Refer to [Authentication](/authentication) for information on how to obtain a
  Bearer token.
</Note>

<Note>
  This endpoint requires the `mobilize.bundle.list` API client authorization
  scope to access the list of bundles. You can set up scopes on the [API
  Clients](https://cloud.belio.co.ke/team-overview/api-access-keys) page.
</Note>

### Rate Limit

This endpoint is rate limited to 1 request per second.
